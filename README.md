# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)


# Getting started
- Clone the repository
```
git clone  https://github.com/himanshupatil363/cpoint5FE.git
```
- Install dependencies
```
cd cpoint5FE
npm install
```
- Build and run the project
```
npm start
```
  Server will start at `http://localhost:3000`
